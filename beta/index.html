Triggers for Start Free Trial Button

  • Hover on
  • Hover off
  • Mouse down
  • Mouse up

Rules

  • Hover on
    • Cursor changes to pointer
    • Button edge lifts slightly (transition: 0.3s ease)
    • Button background color subtly shifts to Mailchimp yellow (#ffe01b)
  • Hover off
    • Cursor reverts to default
    • Button edge resets to original position (transition: 0.3s ease)
    • Button background color returns to white
  • Mouse down
    • Cursor stays as pointer
    • Button edge depresses halfway (transition: 0.2s ease)
    • Background changes to a slightly darker yellow (#ffc300)
  • Mouse up
    • Triggers navigation to a new page
    • Slider animation transitions left (transition: 2s ease-in-out)

Feedback

  • Hover on
    • Cursor changes to pointer
    • Yellow edge of button lifts slightly (transition: 0.3s ease)
  • Hover off
    • Cursor goes back to pointer
    • Yellow edge goes back to button edge
  • Mouse down
    • Cursor remains as pointer
      • WOOOOOOOOOOO
    • Yellow edge depresses halfway while holding (transition: 0.2s ease)
  • Mouse up
    • Releases "pressed" state
    • Initiates slider transition with 2s ease-in-out effect
