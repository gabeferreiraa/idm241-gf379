Triggers for Start Free Trial Button

  • Hover on
  • Hover off
  • Mouse down
  • Mouse up

Rules

  • Hover on
    • Changes cursor to custom dot and outline
    • Cursor expands slightly on hover (transition: 0.15s ease)
    • Yellow edge of button lifts slightly
  • Hover off
    • Cursor reverts to default size
    • Button edge resets
  • Mouse down
    • Cursor contracts slightly to indicate pressing (transition: 0.15s ease)
    • Button edge depresses halfway
  • Mouse up
    • Cursor returns to original size
    • Triggers navigation to a new page with slider transition

Feedback

  • Hover on
    • Custom cursor activates, expanding slightly
    • Yellow edge of button lifts
  • Hover off
    • Custom cursor resets to original size
    • Button edge returns to original position
  • Mouse down
    • Cursor contracts to indicate press
    • Yellow edge depresses halfway
  • Mouse up
    • Cursor returns to full size
    • Slider moves to next page
